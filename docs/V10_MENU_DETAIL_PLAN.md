# GUI's Arc OS v10.0 Menu Detail Plan

v10.0부터는 전체 구조를 유지하고 각 메뉴탭별 완성도를 높입니다.

## 고정 구조

- HOME
- 오늘업무
- 현장소장
- 안전관리
- 품질관리
- 공사관리
- 자재·장비
- DB
- 일정
- 날씨
- 사고·뉴스
- 설정

## 원칙

1. 기존 기능 삭제 금지
2. Guest / 현장등록 / 현장접속 유지
3. TODAY 입력값을 모든 메뉴가 공유
4. 각 역할 메뉴는 섞지 않고 분리
5. DB는 역할별 화면의 근거 데이터로 사용
6. WEATHER는 6일 1시간 강수 비교표 유지
7. 품질관리는 타설 등록 → 공시체 자동생성 → 시험일정 → 결과입력 순서로 고도화
