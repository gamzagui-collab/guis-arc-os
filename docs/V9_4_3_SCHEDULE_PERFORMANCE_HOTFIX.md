# v9.4.3 Schedule Performance Hotfix + Compact Visual UX

- 기준: v9.4.2 실제 수정본
- 현재 버전: v9.4.3

## 핵심 수정
- 일정탭 `renderSchedule()` 호출마다 이벤트가 중복 등록되던 문제를 수정했다.
- `WeakSet`으로 root별 이벤트 바인딩을 1회만 수행한다.
- 날짜를 여러 번 클릭해도 CPU 부하가 누적되지 않도록 했다.

## UI 수정
- 작업/자재/장비/콘크리트 타설 구분을 아이콘 대신 색상 라벨과 좌측 강조선으로 표현한다.
- 공휴일 설명은 달력 내부 공간을 차지하지 않도록 제거하고, 날짜 hover 툴팁으로만 표시한다.
- 우측 상세 카드는 더 낮고 조밀하게 정리했다.
