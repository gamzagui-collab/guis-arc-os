# v9.3.5 Weather Visual Decision Polish

## 변경 목적
날씨 탭의 정보 구조는 v9.3.4를 유지하되, 현장에서 즉시 읽히는 시각 강조와 그래프 표현을 개선했다.

## 반영 사항
- 헤드라인을 `WEATHER · 강수·온도·풍속`으로 단순화.
- `AI 오늘 강수 의견`을 `AI 오늘 날씨 의견`으로 변경.
- AI 의견에 중요 단어 강조 스타일 적용.
- 강수 예보처 4곳의 일치도 기반 신뢰도 의견 추가.
- 오늘 1시간 강수 그래프를 더 세련된 막대 스타일로 개선하고 최고치 스티커 표시.
- 오늘 온도/풍속 조치표는 현장 작업시간 기준 06~18시만 표시.
- 일주일 온도/풍속 흐름은 시간별 전체 데이터를 선그래프로 표시.
- 온도/풍속 그래프 라벨은 6시, 18시, 일별 최고치 위주로 제한.
- 날짜 라벨은 00시 부근에만 표시해 산만함을 줄임.
- 카드 제목에 컬러 포인트 바를 적용해 다른 탭에도 재사용 가능한 카드 제목 디자인 기반 마련.
