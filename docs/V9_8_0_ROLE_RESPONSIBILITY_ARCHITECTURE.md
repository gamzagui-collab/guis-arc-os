# v9.8.0 역할·업무 분류 기준

## 원칙
1. 하나의 Task에는 하나의 주담당(ownerRole)만 둔다.
2. 다른 역할은 협조(supportRoles) 또는 참고(referenceRoles)로 연결한다.
3. 동일 업무를 여러 탭에 복제하지 않는다.
4. 검측 요청·진행·결과관리의 주담당은 공사관리다.
5. 품질관리는 검측에 필요한 시험결과와 품질자료를 제공한다.
6. 장비 서류·자체점검은 자재장비, 작업반경·신호수 등 작업안전은 안전관리다.
