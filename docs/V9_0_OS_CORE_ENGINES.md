# GUI's Arc OS v9.0 OS Core Engines

## 반영

- 브랜드는 GUI's Arc OS로 통일
- WEATHER: 3시간 그래프 제거, 1시간 강수 그래프 적용
- QUALITY: 타설 등록 → 공시체 자동 생성 → 오늘 품질업무 → 공시체 관리대장 구조 적용
- 공시체 기본 템플릿: 120㎥당 30개
  - 수직부재 탈형강도 3개
  - 수평부재 탈형강도 3개
  - 필라서포트 확인 3개
  - 28일 압축강도 표준양생 9개
  - 28일 압축강도 현장양생 9개
  - 예비 공시체 3개
- 품질관리 화면 접힘/펼침 구조 정리
- 레미콘, 슬럼프, 공기량, 염화물, 균열, 재료분리, 양생관리 확장 틀 추가

## 주의

공시체 30개 템플릿은 공공공사/현장 운영 기준에 맞춘 실무 기본안입니다.
최종 적용은 현장 품질관리계획서, 특기시방서, 감리 지시, 최신 기준을 우선합니다.
