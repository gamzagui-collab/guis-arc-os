# GUI's Arc OS v9.8.8

## 간편모드 최종 정리 · 사진 중심 현장행동

### 목적
사용자가 문서를 작성하지 않고 사진을 찍고 확인하는 것만으로 현장 기록을 시작한다.

### 사용자 흐름
1. HOME에서 사진 찍기
2. 오늘 작업 선택
3. 행동 유형 선택
4. 확인하고 저장
5. 확인 또는 완료 처리

### 자동 저장
- Action ID
- Work ID
- 현장 날짜와 촬영 기록시간
- 행동 유형
- 상태 및 확인·완료시간
- 축소된 사진 데이터
- Work Evidence 사진 증빙

### 구조 원칙
- 새로운 독립 DB를 만들지 않고 기존 state와 Evidence 구조를 확장한다.
- Work ID가 선택되면 증빙이 자동 연결된다.
- AI 판독은 아직 연결하지 않으며 다음 사진 AI 단계에서 Worker/API를 통해 확장한다.
- AI 결과도 사용자의 최종 확인을 거친 뒤 업무와 증빙으로 확정한다.
