# GUI's Arc OS v9.3.4 Weather Layout Redesign

## 목적

v9.3.4는 날씨 탭의 강수·온도·풍속 정보 배치를 실제 현장 관리자 사용 흐름에 맞게 재정렬한 버전이다.

## 반영 내용

1. 강수량은 오늘 1시간 그래프와 6일 시간별 강수 비교표를 연속 배치했다.
2. 온도는 오늘 1일차 시간별 조치표를 먼저 보여주고, 일주일 정보는 흐름 그래프로 단순화했다.
3. 풍속도 오늘 1일차 시간별 조치표를 먼저 보여주고, 일주일 정보는 최대 풍속 흐름 그래프로 단순화했다.
4. 온도·풍속 조치표의 반복 문장 안에서 작업중지, 열사병, 11~15시, 크레인, 고소작업, 신호수, 작업반경 등 주요 단어를 색상 강조했다.
5. 강수 일치도 색상은 별 1~2개 빨강, 3개 주황, 4~5개 초록으로 유지했다.

## 설계 기준

- 강수: 비가 언제 오는지와 예보처 편차를 빠르게 확인한다.
- 온도: 오늘 시간대별 온열·한랭 조치 판단을 우선한다.
- 풍속: 오늘 시간대별 크레인·고소·외장 작업 주의/중지 판단을 우선한다.
- 일주일 온도·풍속: 세부 표보다 날짜별 위험 흐름만 확인한다.
