# GUI's Arc OS v8.8 Role Separated OS

## 변경

- 메뉴를 GUI's Arc 방식에 가깝게 역할별로 재분리
- 현장소장 / 안전관리 / 품질관리 / 공사관리 / 자재·장비관리 페이지 추가
- 업무가이드와 AI 현장비서의 중복을 역할별 오늘 업무로 흡수
- KNOWLEDGE는 단순 요약이 아니라 공종·품질·안전·기준 DB로 유지
- WEATHER에 3시간 강수 그래프 추가

## 방향

기능을 무리하게 통합하지 않고, 현장 업무 역할별로 분리합니다.
DB는 계속 확장하고, 각 역할 페이지는 TODAY 입력값을 바탕으로 필요한 내용만 표시합니다.
