# 새 채팅 시작 프롬프트

GUI's Arc OS 공식 개발 프로젝트를 이어서 진행한다.

현재 기준 버전은 v9.3.2 Final Handoff이며, 프로젝트 기준 문서는 `/docs/GUI_Arc_OS_MasterPlan_v10.md`이다.

GUI's Arc OS는 건설현장 운영체제(Construction Site Operating System)로, 현장소장·안전관리·품질관리·공사관리·자재/장비관리를 하나의 OS에서 지원하는 장기 프로젝트이다.

목표는 정보 조회 사이트가 아니라, 관리자가 오늘 무엇을 해야 하는지 판단하도록 돕는 OS를 만드는 것이다.

현재까지의 핵심 구조:

- Guest 시작
- 현장 등록
- 현장 접속
- HOME
- 오늘업무
- 현장소장
- 안전관리
- 품질관리
- 공사관리
- 자재·장비
- DB
- 일정
- 날씨
- 사고·뉴스
- 설정

v9.3.2에서 반영된 최종 날씨 방향:

- 헤더와 메뉴는 한 줄 통합
- 모바일은 햄버거 메뉴
- 6일 1시간 강수 비교표
- 1시간 강수 그래프
- KMA / ECMWF / GFS / JMA 비교
- 예보처 4곳은 동일 하늘색 계열
- 평균/최대/일치도 열 구분
- 날짜는 7/9 + (수) 형식
- 좌표 선택 지도는 Windy 좌표 이동 목적
- 지도 클릭 시 Windy 지도도 동일 좌표로 이동
- AI 오늘 강수 의견은 그래프 위 전체 폭 카드

앞으로는 v9.4부터 Today Engine을 완성하고, 이후 v9.5 현장소장, v9.6 안전관리, v9.7 품질관리, v9.8 공사관리, v9.9 자재·장비, v10.0 통합 순서로 진행한다.

기존 기능은 삭제하지 말고, 기능 추가 방식으로 개발한다.

## v9.3.3 Weather Decision Expansion
- 이번 채팅에서 v9.3.3으로 날씨 탭을 확장했다.
- AI 오늘 강수 의견의 기준시간은 KST 현재 날짜/시간으로 표시한다.
- 날씨 탭의 콘크리트 타설 KPI와 별도 콘크리트 타설 판단 카드는 제거했다.
- AI 오늘 강수 의견에 야외작업/실내작업/풍속/온도 판단을 통합했다.
- 6일 시간별 강수 비교표 일치도 색상을 별 개수별로 빨강/주황/초록 처리했다.
- 강수 비교표 스크롤 높이를 키워 하루 단위 확인 가독성을 개선했다.
- 강수 아래에 일주일 온도 그래프/온도 비교표, 일주일 풍속 그래프/풍속 비교표를 추가했다.


## v9.3.4 Weather Layout Redesign
- 날씨 탭은 v9.3.4 기준으로 강수/온도/풍속 정보 배치를 재정렬했다.
- 강수는 오늘 그래프 다음 6일 비교표를 연속 배치한다.
- 온도는 오늘 시간별 조치표 다음 일주일 최고 체감온도 흐름 그래프를 배치한다.
- 풍속은 오늘 시간별 조치표 다음 일주일 최대 풍속 흐름 그래프를 배치한다.
- 온도/풍속 조치표는 주요 단어를 색상 강조하여 반복 문장 속에서도 핵심 조치를 빠르게 보도록 했다.

## v9.3.5 Weather Visual Decision Polish
- 날씨 탭 헤드라인을 `WEATHER · 강수·온도·풍속`으로 단순화했다.
- `AI 오늘 날씨 의견`으로 명칭을 변경하고, 중요 단어 시각 강조 및 예보처 4곳 일치도 기반 신뢰도 의견을 추가했다.
- 오늘 1시간 강수 그래프는 세련된 막대형 디자인으로 개선하고 최고치 스티커를 추가했다.
- 오늘 온도/풍속 조치표는 06~18시만 표시한다.
- 일주일 온도/풍속 흐름은 시간별 선그래프로 표시하며 6시/18시/일별 최고치/00시 날짜 라벨만 노출한다.
- 카드 제목에는 컬러 포인트 바를 적용해 타 탭 카드에도 재사용 가능한 제목 디자인 기반을 마련했다.

## v9.3.6 Weekly Weather Graph Callout Patch
- 날씨 탭 일주일 온도/풍속 흐름 그래프 개선
- X축 시간 라벨: 00시, 06시, 12시, 18시 기준 표시
- 일별 최고치는 X축이 아니라 그래프 위 점에 말풍선으로 표시
- 최고치 표기 형식: `최고치[17시]`
- 온도 최고치 말풍선은 붉은 계열, 풍속 최고치 말풍선은 연두/녹색 계열
