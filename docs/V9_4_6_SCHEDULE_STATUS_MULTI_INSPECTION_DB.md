# v9.4.6 Schedule Status / Multi Drag / Inspection DB

## 변경 요약
- 상태 수정 UI를 타이핑 방식에서 선택식으로 변경했다.
- 선택 날짜의 일정 전체삭제를 추가했다.
- 여러 카드 선택 후 날짜 드롭으로 일괄 이동할 수 있다.
- 점검 유형은 공종 DB에 `점검`으로 정식 등록했다.

## 점검 DB
- 국토안전부
- 안전보건공단
- 노동부

점검 일정은 향후 각 관리자 업무에 연결한다. 예를 들어 국토부 점검이 잡히고 현재 철근콘크리트공사가 주공정이면, 배근검측, 콘크리트 타설기록, 공시체, 강도관리, 동바리 존치자료 등을 우선 준비 항목으로 제안할 수 있다.
