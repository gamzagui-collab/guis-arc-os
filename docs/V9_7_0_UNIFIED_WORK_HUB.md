# GUI's Arc OS v9.7.0 — Unified Work Hub

## 목적
동일 Work ID에 연결된 안전, 품질, 공사, 자재·장비, 증빙 및 결재 상태를 한 화면에서 확인한다.

## 원칙
- 새로운 업무 DB를 만들지 않는다.
- Work Instance, Task, 품질, 공사, 자원, 증빙 저장소를 조회하여 통합한다.
- 원본 업무의 완료 여부를 허브에서 임의 변경하지 않는다.
- 수정은 각 담당 관리 탭에서 수행한다.

## 주요 기능
- 날짜별 Work Instance 목록
- 누락 자동검사 상태 및 대응상태
- 품질 체크, 검측, 공시체 현황
- 공사 체크, 도면 확인, 인수인계 현황
- 자재·장비 준비 및 장비 점검 현황
- Work ID별 증빙 목록
- 담당 탭 바로가기
