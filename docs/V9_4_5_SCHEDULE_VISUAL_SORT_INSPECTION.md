# GUI's Arc OS v9.4.5 Schedule Visual Sort + Inspection Type

## Purpose
일정탭을 실제 현장 일정판에 가깝게 정리한다.

## Changes
- 달력의 타설 표시 색상을 상세 카드와 통일.
- 카드 정렬 버튼 추가. 기본 순서는 타설 → 점검 → 작업 → 장비 → 자재이며 버튼 재클릭 시 역순.
- 달력 칩의 글자 겹침 방지를 위해 고정 높이, 말줄임, 내부 간격 조정.
- 상세카드에서 안전/품질/공사 프리뷰 제거.
- 점검 유형 추가. 안전공단, 국토안전부, 감리, 발주처 점검을 일정으로 등록하고 이후 관리자별 준비사항 제안과 연결한다.

## Design Rule
세부공종은 여전히 기준 DB 선택 구조를 유지한다. 단, 상세카드에는 공종 기준 설명을 직접 노출하지 않고 필요한 화면/관리자 업무에서만 호출한다.
