# GUI's Arc OS v9.8.3 — 품질관리 간편화

## 목적
품질관리자의 주담당 업무를 시험, 공시체, 품질자료, 품질사진으로 명확히 하고 검측·시공관리는 공사관리 참고정보로 분리한다.

## 사용자 흐름
1. 일정 또는 오늘업무에 공정을 한 번 등록한다.
2. 시스템이 필요한 품질시험과 품질자료 항목을 자동 생성한다.
3. 품질관리자는 실시 또는 확인 체크만 한다.
4. 사진은 선택 즉시 Work ID에 연결된다.

## 자동 생성
- 콘크리트: 슬럼프, 공기량, 염화물, 콘크리트 온도
- 공통 품질자료: 시험성적서, 자재승인서·납품서류

## 역할 분리
- 검측 요청·결과: 공사관리 주담당
- 시험·공시체·품질자료: 품질관리 주담당
- 검측상태는 품질관리에서 읽기 전용 참고정보

## 데이터
- 기존 Work ID와 완료이력 유지
- `qualityDocuments`를 통합 상태에 확장
- 데이터 스키마 21
