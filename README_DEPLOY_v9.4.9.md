# GUI's Arc OS v9.4.9 Stable 배포 안내

이 ZIP은 사용자가 검증한 v9.4.9 백업본을 기준으로 문서/버전 기준을 정리한 Stable 패키지입니다.

## 적용 방법

1. 현재 사용 중인 `guis-arc-OS-enterprise` 폴더를 복사해서 백업합니다.
2. 이 ZIP을 압축 해제합니다.
3. 압축 해제된 파일 전체를 기존 `guis-arc-OS-enterprise` 폴더에 덮어씁니다.
4. PowerShell에서 프로젝트 루트로 이동합니다.
5. 버전 동기화 후 배포합니다.

```powershell
npm run sync:version
npm run deploy:pages
```

## 배포 후 확인

- 헤드라인 버전이 `v9.4.9`인지 확인합니다.
- 일정탭에서 날짜를 여러 번 클릭해도 CPU 부하가 증가하지 않는지 확인합니다.
- 일정 카드 드래그 이동이 되는지 확인합니다.
- 체크 후 묶음 이동/선택 삭제가 되는지 확인합니다.
- 수정창이 가로 스크롤 없이 표시되는지 확인합니다.
- 확정/미확정 뱃지가 자연스럽게 우측에 정렬되는지 확인합니다.

## v9.5.0 개발 전 기준

v9.4.9는 일정탭 Stable 기준입니다. 다음 개발은 v9.5.0에서 `일정 → 관리자 업무 자동 반영`으로 진행하는 것을 권장합니다.
